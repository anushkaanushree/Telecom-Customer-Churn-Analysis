# Telecom Customer Churn – Revenue Impact & Retention Strategy

## Project Overview
This project analyzes telecom customer churn with a focus on financial impact and retention strategy.

##  Objectives
- Identify key drivers of churn
- Quantify revenue impact
- Calculate customer lifetime value (CLV)
- Identify high-risk customer segments
- Model retention improvement scenarios

##  Key Insights
- Overall churn rate: 26.5%
- Annual revenue lost due to churn: ₹1.66M
- High-risk segment (Fiber optic + Month-to-month) churn: ~55%
- Revenue lost from high-risk segment: ₹1.73M
- 3% churn reduction could recover ₹189K annually

##  Tools Used
- Python (Pandas, Matplotlib)
- Data Analysis


##  Business Impact
This analysis demonstrates how churn directly affects recurring revenue and highlights strategic retention opportunities.
